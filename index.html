About the Project

The inspiration for this project came from a personal experience where timely health monitoring could have made a significant difference. Understanding the potential of AI in the healthcare sector, we embarked on creating a system that not only monitors health metrics in real-time but also predicts potential issues, providing peace of mind and preventive care.

This project is part of our portfolio for Holberton School. You can find more about it on our GitHub repository.
